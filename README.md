Mano, which translates directly to "Hand" across many cultures, is designed to act as a seamless, minimal extension of the user rather than a cold, unapproachable machine. Built as a native Android application using Java, the app focuses on an accessible, high-end design pattern that eliminates visual noise while remaining completely functional. Upon a three-second launch sequence handled by the splash screen to cleanly configure window bindings and EdgeToEdge layout processing, users are brought to a secure authentication interface. This setup instantly sanitizes inputs by stripping empty trailing spaces before verifying or creating credentials locally inside an on-device SQLite database ledger named ManoUser.db



The core user experience is centered around the dialog screen, which uses a high-performance RecyclerView to instantly map user inputs into chat bubbles. While an asynchronous, non-blocking network background request is safely fired over OkHttp to communicate with the Groq Cloud API gateway running llama3-8b-8192, an inline spinning loading indicator keeps tracking the active system state. Once the API handles payload serialization and delivers the incoming message string, the loading state clears, and the AI's reply bubble is immediately rendered on the left. Users can also interact with a premium bottom sheet layout designed under Material Design 3 guidelines to access introductory FAQs, fire open an explicit intent wrapper connecting to the device's native mail client for technical support, or clear their entire message history directly from the local SQLite layer via a warning-state red typography button.  

To make reviewing this project as straightforward as possible, all comprehensive presentation material and workflow visuals have been uploaded directly to the repository as App Overview (Version 1).pdf. Additionally, the fully compiled executable file, mano apk, is readily available within the root of the repository, allowing anyone to immediately download, install, and experience the Mano AI assistant firsthand
